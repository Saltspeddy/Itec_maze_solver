# Maze Solver Robot for ITEC#18

This repository contains code snippets used in developing a maze solver robot for the ITEC#18 competition. The robot was built in just 44 hours as part of the challenge.

## Robot Overview

A simple yet effective maze-solving robot built within tight time constraints, demonstrating efficient problem-solving and rapid prototyping.

## Hardware Components

The main components used in this project include:

- **Sensors**:
  - HC-SR04P ultrasound sensors (for distance measurement)

- **Motors & Control**:
  - 2 standard DC motors
  - L298N motor driver

- **Power Management**:
  - XL6009 boost voltage module

- **Connectivity**:
  - HC-05 Bluetooth module

- **Prototyping**:
  - 1 Breadboard

- **Main Controller**:
  - STM32F407G-DISC1 development board

## Project Details

- **Development Time**: 44 hours
- **Competition**: ITEC#18
- **Main Challenge**: Maze navigation and solving
